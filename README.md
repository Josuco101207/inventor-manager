# Inventor Manager Pro

Sistema profesional de gestión de inventarios y préstamos de herramientas.

## Características
- Gestión en tiempo real con Firebase.
- Soporte Offline y PWA.
- Escaneo de códigos QR.
- Dashboard de métricas y análisis.
- Control de mantenimiento y fallas.
